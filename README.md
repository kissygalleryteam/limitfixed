### V1.0

* 在指定容器中设置元素fixed浮动显示。
* 支持横向和纵向的组合浮动
* 支持相对父容器的偏移显示
* 支持构建占位元素和自定义样式
* 支持IE6下fixed浮动。
* 支持事件触发
